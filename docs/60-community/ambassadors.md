## Who this is for

You want to help Acurast grow by teaching others: writing tutorials, making videos, hosting meetups, and supporting newcomers.

## Official reference (primary)

Start here:

- Ambassador program: `https://acurast.com/ambassador-program/`
- Blog: `https://acurast.com/blog/`
- Docs: `https://docs.acurast.com/`

## What to create (beginner-friendly)

- **Provider onboarding content**: Lite/Core setup walkthroughs with screenshots.
- **Developer labs**: “Deploy your first app”, “Secrets”, “Replicas”, “Troubleshooting”.
- **Use-case explainers**: why decentralized compute matters; real-world examples.

## Quality checklist

- Cite official sources and link them.
- Use safe screenshots (no keys/mnemonics).
- Include prerequisites, expected time, and troubleshooting.
- Make it accessible: captions for videos, alt text for diagrams.

## How we use this repo

- Tutorials and docs live in version control for review.
- Use `CONTRIBUTING.md` for writing conventions and PR checklist.
- Prefer issues for proposals; PRs for concrete improvements.
