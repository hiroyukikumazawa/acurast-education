## Welcome

`acurast-education` is an open-source curriculum to help you learn Acurast from **beginner → provider → developer → community creator**.

## What is Acurast? (in one sentence)

Acurast is a **decentralized compute network** where smartphones can run deployments for developers, coordinated by an on-chain orchestration layer. Start from the official docs: `https://docs.acurast.com/`.

## Choose your path

- **I’m new**: Read `docs/00-introduction/key-concepts.md`.
- **I want to provide compute**: Go to `docs/10-providers/getting-started.md`.
- **I want to use Hub / wallets**: Go to `docs/50-hub/wallets-and-hub.md`.
- **I want to deploy apps**: Start with:
  - “Instant deploy” in Hub: `docs/20-developers/instant-deploy-hub.md`
  - “Deploy via CLI”: `docs/20-developers/deploy-first-app-cli.md`
- **I want tokenomics / staking**: Go to `docs/30-tokenomics/intro.md` and `docs/40-staked-compute/overview.md`.
- **I’m doing Cloud Rebellion quests**: Go to `docs/14-cloud-rebellion/overview.md`.
- **I want to teach others**: Go to `community/README.md` and `docs/60-community/ambassadors.md`.
