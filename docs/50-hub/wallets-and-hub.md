## Goal

Understand what the Acurast Hub is and how wallet access fits into common flows (provider onboarding and developer deployments).

## Hub (official)

The Hub is available at `https://hub.acurast.com/`.

You’ll use it to:

- onboard processors (Lite/Core)
- view activity (deployments, executions)
- manage relevant ecosystem actions (depending on network and features)

## Wallet basics

Wallets are used to sign transactions and interact with Acurast network tools.

For developer workflows, the official “First App Deployment” tutorial notes you can import the CLI-generated mnemonic/account into a wallet (e.g. Talisman) to use the same address in the web console:
`https://docs.acurast.com/developers/deploy-first-app/`
