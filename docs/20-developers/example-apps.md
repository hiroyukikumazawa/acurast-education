## Official example apps (primary)

Acurast maintains a set of example deployments you can clone and modify:
`https://github.com/Acurast/acurast-example-apps`

Start with the official developer tutorial references:

- Fetch API example: `https://github.com/Acurast/acurast-example-apps/tree/main/apps/app-fetch`
- Express webserver example: `https://github.com/Acurast/acurast-example-apps/tree/main/apps/app-webserver`

## Suggested labs (beginner → advanced)

- **Lab 1 (Beginner)**: Deploy the fetch example and log output.
- **Lab 2 (Beginner)**: Deploy the webserver example, verify remote URL.
- **Lab 3 (Intermediate)**: Add secret env vars via `includeEnvironmentVariables`.
- **Lab 4 (Intermediate)**: Increase replicas; compare execution behavior.
- **Lab 5 (Advanced)**: Build a small “oracle” style workflow (fetch → sign → publish result).
