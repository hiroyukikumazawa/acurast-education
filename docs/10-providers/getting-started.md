## Who this is for

You want to turn a phone into an Acurast processor and start providing compute.

## Choose Lite vs Core

Official reference: `https://docs.acurast.com/acurast-processors/`

- **Lite**: quickest start (Android + iOS), no factory reset.
- **Core**: dedicated Android device, factory reset + locked down environment.

## Requirements (official)

Processor requirements and links are documented here:
`https://docs.acurast.com/acurast-processors/`

## Setup steps (high-level)

Follow the official “Become a Compute Provider” guide:
`https://docs.acurast.com/processors/become-compute-provider/`

This curriculum adds a practical checklist:

- **Power**: keep the phone charging during long runs.
- **Connectivity**: stable Wi‑Fi is preferred.
- **Security**: never expose wallet secrets; use a dedicated device for Core.
