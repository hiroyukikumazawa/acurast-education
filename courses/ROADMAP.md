# Course roadmap

This roadmap organizes the curriculum into structured courses from beginner → advanced.

## Course 0: Acurast Fundamentals (Beginner)

- **Outcome**: explain Acurast at a high level and navigate official resources confidently.
- **Modules**
  - `docs/00-introduction/overview.md`
  - `docs/00-introduction/key-concepts.md`
  - `docs/50-hub/wallets-and-hub.md`

## Course 1: Become a Compute Provider (Beginner → Intermediate)

- **Outcome**: set up Lite or Core, connect to Hub, keep stable uptime.
- **Modules**
  - `docs/10-providers/getting-started.md`
  - `docs/10-providers/troubleshooting.md`
  - Official references:
    - `https://docs.acurast.com/acurast-processors/`
    - `https://docs.acurast.com/processors/become-compute-provider/`

## Course 2: Deploy on Acurast (Intermediate)

- **Outcome**: deploy a bundled Node.js app via CLI; iterate using examples.
- **Modules**
  - `docs/20-developers/deploy-first-app-cli.md`
  - `docs/20-developers/example-apps.md`
  - Official reference: `https://docs.acurast.com/developers/deploy-first-app/`

## Course 3: Economics (Intermediate)

- **Outcome**: understand ACU token utility, allocations, and inflation.
- **Modules**
  - `docs/30-tokenomics/intro.md`
  - Official reference: `https://docs.acurast.com/tokenomics`

## Course 4: Staked Compute (Advanced)

- **Outcome**: understand committers vs delegators, rewards, and slashing risk.
- **Modules**
  - `docs/40-staked-compute/overview.md`
  - Official reference: `https://docs.acurast.com/staked-compute/overview`

## Course 5: Community Creator / Ambassador (All levels)

- **Outcome**: create high-quality educational content and run community workshops.
- **Modules**
  - `docs/60-community/ambassadors.md`
  - `community/README.md`
  - Official reference: `https://acurast.com/ambassador-program/`
