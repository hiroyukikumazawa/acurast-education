# acurast-education

An open-source education platform for learning **Acurast** — from first steps (what Acurast is) to becoming a **compute provider**, shipping **apps on Acurast Cloud**, and contributing as a **community creator / ambassador**.

## What’s inside

- **`docs/`**: The main curriculum (beginner → advanced).
- **`tutorials/`**: Hands-on, copy/paste friendly walkthroughs.
- **`courses/`**: Structured learning paths and roadmaps.
- **`videos/`**: Video tutorial scripts + shot lists.
- **`assets/`**: Diagram descriptions, visuals, and templates.
- **`community/`**: Ambassador/community playbooks and governance/contribution resources.
- **`website/`**: A Docusaurus site that publishes `docs/`.

## Quickstart (local docs site)

Prereq: Node.js 20+

```bash
cd website
npm install
npm run start
```

Then open the printed local URL.

## How to use this repo

- **If you’re new**: start at `docs/00-introduction/overview.md`.
- **If you want to provide compute**: go to `docs/10-providers/`.
- **If you want to build apps**: go to `docs/20-developers/`.
- **If you want tokenomics / staking**: go to `docs/30-tokenomics/` and `docs/40-staked-compute/`.
- **If you want to contribute**: read `CONTRIBUTING.md` and `community/README.md`.
