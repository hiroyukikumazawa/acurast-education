## Goal

Show the fastest successful deployment path: Hub → Create Deployment → IPFS URL → Publish → Verify via `.acu.run`.

## Primary reference (official)

`https://docs.acurast.com/developers/deploy-first-app-console`

## Script

### Scene 1 — What we’re doing (0:00–0:20)

- **On-screen**: “Deploy in minutes”
- **Voiceover**: “In the next few minutes you’ll publish a working Hello World deployment on Acurast using the Hub.”

### Scene 2 — Create deployment (0:20–2:00)

- **B‑roll**: Hub → Create Deployments → Deployment Code → IPFS URL
- **On-screen text**: paste `ipfs://Qmb7sw1mH349wNJsoGWRs1HedAwVXtHPsW4YoXok15DeZL`
- **Voiceover**: “Paste the prepared IPFS bundle, keep defaults unless you want to select a specific processor.”

### Scene 3 — Schedule + publish (2:00–3:15)

- **B‑roll**: One-time execution, end time in future, Suggest Reward, Publish
- **Voiceover**: “Set a one-time execution schedule and publish.”

### Scene 4 — Verify (3:15–end)

- **B‑roll**: Open Deployments → Assigned Processors → copy Main Acurast Account
- **On-screen**: `https://<processor-address>.acu.run`
- **Voiceover**: “Once it starts, open the `.acu.run` URL using the processor address from the Assigned Processors section.”
