# Workshops

Workshops are guided, time-boxed sessions designed to reliably get beginners to a “first win”.

## Available workshops

- **Workshop A (60 min)**: Acurast overview + Instant deploy via Hub
- **Workshop B (90 min)**: Deploy via CLI + iterate on an example app
- **Workshop C (60 min)**: Community content sprint (ship a PR)

## Facilitator checklist

- Confirm Wi‑Fi and projection
- Share a link list at start (docs, hub, repo)
- Remind: **never share mnemonics/private keys**
- End with a “next steps” slide + contribution CTA
