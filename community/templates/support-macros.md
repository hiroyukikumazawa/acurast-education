# Support macros (copy/paste replies)

Use these to help newcomers quickly while staying safe.

## Safety reminder (always)

“Please don’t share mnemonics/private keys or screenshots that contain sensitive wallet information.”

## Instant deploy via Hub

“Try the official Hub tutorial: `https://docs.acurast.com/developers/deploy-first-app-console`. If verification fails, open Hub → Deployments → your deployment → Assigned Processors and copy the processor address (lowercase), then open `https://<processor-address>.acu.run` once the start time is reached.”

## Provider setup (Lite/Core)

“Start with the official processor overview and provider guide: `https://docs.acurast.com/acurast-processors/` and `https://docs.acurast.com/processors/become-compute-provider/`. If you share what device/OS you’re on (no secrets), we can narrow it down.”

## CLI deploy

“Follow `https://docs.acurast.com/developers/deploy-first-app/`. If `acurast deploy` says balance is 0, use the faucet link it prints, then run deploy again. Don’t commit `.env`.”
